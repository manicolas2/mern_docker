# mern_docker
Template for a MongoDB/MySQL-ExpressJS-ReactJS-NodeJS (M-E-R-N) Application Stack using Docker
 (Tested on Ubuntu 16.04 x86_64)

* Install docker
* Install latest docker-compose
* Install NodeJS via NVM
* docker-compose build
* docker-compose up
